# Student Record System

A Python console application that allows users to manage student records.

## Features

- Add Students
- View Students
- Search Students
- Show Top Student
- Calculate Average Grade

## Technologies Used

- Python
- Functions
- Lists
- Dictionaries
- Match Case

## How to Run

```bash
python main.py
```
